🌐 Acronymia: The Global Dictionary of Short & Full Forms

Acronymia is a premium, fully interactive single-page web application designed to help users search, decode, and learn abbreviations and acronyms from all over the world. It features a curated offline database across major industries, a robust interactive quiz system, and a live AI-powered fallback search powered by Google's Gemini 2.5 Flash API to decode any obscure short form in real-time.

🚀 Features

🔍 Intelligent Search & Autosuggest: Instantly filter over 100 pre-seeded acronyms spanning Technology, Medicine, Finance, Slang, Science, and Global Organizations.

✨ Live AI Lookup: Can't find an acronym in the offline dictionary? Click "Lookup Globally with AI" to query Gemini 2.5 Flash for the full definition, category, and real-world usage examples dynamically.

🏆 Interactive Quiz Arena: Test your knowledge! Choose your preferred sector, customize the number of questions, and compete to get a high score.

➕ Community Contributions: Add your own acronyms directly to the session database, complete with descriptions and usage contexts.

🌓 Adaptive Dark/Light Mode: Seamlessly switches themes according to your system preference or via manual toggle.

📱 Fully Responsive Design: Clean, modern, touch-optimized fluid layout crafted with Tailwind CSS for mobile, tablet, and desktop screens.
